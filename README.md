## :checkered_flag: Starting

```zsh
# Clone this project
$ git clone https://github.com/uni-clouds/unicloud-frontwebapp

# Access
$ cd unicloud-app

# Install dependencies
$ yarn or npm install

# Run the project
$ yarn start or npm run preview

# The server will initialize in the <http://localhost:3000>
```
